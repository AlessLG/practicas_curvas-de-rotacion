# Ajuste a curvas de rotacion

Proyecto de año de prácticas

Universidad San Carlos de Guatemala

Utilizando principalmente
- Mínimos cuadrados
- Inferencia bayesiana con Montecarlo vía cadenas de Markov

Se encuentran los parámetros de las funciones de densidad de Plummer, y NFW para ajustarse a los datos de las curvas de rotación de distintas galaxias de bajo brillo superficial (LSB).
